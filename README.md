# Smart Crop Monitoring System (Simulation-Based IoT)

## Overview
A simulation-based IoT crop monitoring system that uses environmental and soil parameters to recommend suitable crops and monitor irrigation requirements.

## Features
- Crop Recommendation using Machine Learning
- Simulated IoT Sensor Inputs
- Irrigation Monitoring Logic
- Soil Parameter Analysis
- Future ESP32/Arduino Sensor Integration
- Gradio Dashboard Interface

## Input Parameters
- Temperature
- Humidity
- Soil Moisture
- pH
- Nitrogen
- Phosphorus
- Potassium

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Gradio

## Future Scope
The system is designed for future integration with real IoT sensors such as ESP32, Arduino, DHT11, soil moisture sensors, and cloud monitoring platforms.
